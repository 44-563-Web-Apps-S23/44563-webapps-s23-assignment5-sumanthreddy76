The row with the plant names breaks first when I gradually shrink the browser size. This is due to the fact that the plant names are in larger divs with class col-md, which use more space than the smaller divs with class col-sm-2, which simply contain the letters. The larger divs are shifted to the next row as the screen size decreases since there is not enough room for them. On the other hand, the smaller letter-only divs can occupy a smaller area and remain on the same row for a longer period of time.
